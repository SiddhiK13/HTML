Document 1-Lists:Unordered (<ul>) & Ordered (<ol>)

Purpose (simple)

Lists help you group related items. Use them when you have a set of things (steps, features, ingredients, links).

Unordered list (<ul>) - bullet points

  • Used when order does not matter (features, items, tags).
  • Browser shows bullets by default.

Basic syntax

<ul>
   <li> First item</li>
   <li> Second item</li>
<ul>

Example: Shopping list

<ul>
   <li> Milk <li>
   <li> Bread <li>
   <li> Eggs <li>
<ul>
